Welcome to the Escape the Valley handbook. This guide covers installation, gameplay mechanics, GM profiles, platform support, and troubleshooting.

## What is Escape the Valley?

Escape the Valley is an Oregon Trail-style survival game that runs in your terminal. Lead a party of settlers through a procedurally generated wilderness. Manage food, water, wagon condition, and morale while navigating events, hazards, and hard choices.

The valley is harsh but fair. A skilled player wins about 1 in 3 runs.

## Key features

- **Zero setup** -- one `npx` command downloads a platform binary, verifies SHA256, and caches it locally
- **AI Game Master** -- three storytelling voices powered by Ollama, running entirely on your machine
- **Procedural world** -- seeded generation makes every run reproducible and every playthrough different
- **XRPL Ledger Backpack** -- optional on-chain inventory tracking on XRPL Testnet
- **Cross-platform** -- pre-built binaries for Linux x64, macOS ARM64, and Windows x64
